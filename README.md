# 💳 Alke Wallet

> Primer proyecto del Bootcamp **Desarrollo de Aplicaciones Full Stack Python** — una billetera digital construida con HTML, CSS y JavaScript.

---

## 📋 Descripción

**Alke Wallet** es una aplicación web de billetera digital que permite a los usuarios gestionar su dinero de forma sencilla: iniciar sesión, consultar su saldo, depositar fondos, enviar dinero a otros usuarios y revisar su historial de transacciones.

Este proyecto fue desarrollado como entrega final del primer módulo del Bootcamp Full Stack Python, aplicando conceptos fundamentales de desarrollo frontend: estructura HTML semántica, estilos CSS responsivos y lógica de validación con JavaScript.

---

## 🚀 Características principales

- 🔐 **Inicio de sesión** — pantalla de login con validación de credenciales
- 🏠 **Menú principal** — panel de navegación con acceso a todas las funciones
- 💰 **Depósito de fondos** — formulario para ingresar dinero a la cuenta
- 📤 **Envío de dinero** — transferencias a otros usuarios con validaciones
- 📄 **Historial de transacciones** — registro visual de movimientos realizados
- ✅ **Validaciones JavaScript** — control de errores en formularios en tiempo real

## ▶️ Cómo ejecutar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/veragonzalo/Proyecto_Alke_Wallet.git
```

2. Navega a la carpeta del proyecto:

```bash
cd Proyecto_Alke_Wallet
```

3. Abre el archivo `index.html` en tu navegador:

```bash
# En macOS
open index.html

# En Linux
xdg-open index.html

# En Windows
start index.html
```

> No requiere instalación de dependencias ni servidor backend. Es una aplicación 100% frontend.
